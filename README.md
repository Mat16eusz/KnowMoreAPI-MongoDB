# KnowMoreAPI-MongoDB
## NodeJS + MongoDB API KnowMore


## Usage
1. Clone the repository.
```
https://github.com/Mat16eusz/KnowMoreAPI-MongoDB.git
```
2. Open the folder in terminal and install package:
```
npm init
npm install express mongoose
```
